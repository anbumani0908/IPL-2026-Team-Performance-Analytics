An end-to-end **Cricket Analytics Project** built using **Power BI**, **Python**, **Power Query**, and **DAX** to analyze the performance of all IPL teams during the **IPL 2026 season**.

The project starts with **ball-by-ball Cricsheet data**, transforms it into an analytics-ready dataset using Python, and presents interactive dashboards with team-wise insights and visual storytelling.

---

# 📌 Project Overview

This project demonstrates the complete analytics workflow:

# 📈 Key Insights

- Analyzed **10 IPL teams** using ball-by-ball data.
- Processed thousands of deliveries using Python.
- Created multiple DAX measures for team and player statistics.
- Designed team-specific dashboards using official team themes.
- Built reusable Power BI templates for all teams.
- Presented batting, bowling, captain impact, and phase-wise scoring insights.
